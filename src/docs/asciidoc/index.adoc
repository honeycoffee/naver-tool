=== 데이터 스키마 정보

- 조회성 API(목록, 상세정보, 데이터 스키마 조회)에 대해서만 데이터 검색/입력 폼 참고 용도로 제공한다.

- Select Box, Radio Button, CheckBox 등 미리 정의된 코드 목록이 필요한 필드의 경우 코드목록 데이터 정보를 포함하여 제공한다.
